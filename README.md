chrome-ext-my-hatebu-search-in-google
======================

概要 / Overview
----------
マイブックマークをアイコンから参照でるようにするChrome Extensionです。

Google検索結果画面では現在の検索ワードに一致するブックマーク件数がバッジ表示され、アイコンをクリックすると絞り込まれた状態でマイブックマークが表示されます。
